# Can Boss RPG

11 yaşındaki Can'ın tasarladığı 2D top-down RPG: bir odada uyan, WASD ile dolaş, boss'larla matematik sorularıyla savaş.

## Lokal Çalıştırma

```bash
cd can-boss-rpg
python3 -m http.server 8000
```

Tarayıcıda → `http://localhost:8000`
