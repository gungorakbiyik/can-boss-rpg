# Feature Listesi + Sprint Planı

## v1 — MVP (Sprint 1-7)

### Must Have
- [ ] Phaser scaffold + GitHub Pages deploy (Sprint 1)
- [ ] Oda + WASD karakter hareketi (Sprint 1)
- [ ] 3 etkileşim zone'u — ısınma, mağaza, boss kapısı (Sprint 2)
- [ ] Isınma sorusu sahne — basit math, DOM modal (Sprint 3)
- [ ] Boss savaşı split-screen — HP barları, 4 soru, hasar (Sprint 4)
- [ ] Yanlış cevap sona kalır (Sprint 4)
- [ ] Coin kazanma — level başına (Sprint 5)
- [ ] Mağaza — +HP, ipucu (Sprint 5)
- [ ] Level progression (Sprint 6)
- [ ] localStorage save (Sprint 6)
- [ ] Bug fix + cila (Sprint 7)

### Should Have
- [ ] Karakter sprite (emoji default, sonra Can'ın seçimi)
- [ ] Boss farklı görseli her level
- [ ] Hasar animasyonu (shake)
- [ ] Doğru cevap ses efekti

### Could Have
- [ ] Müzik (background)
- [ ] Cutscene (boss tanıtımı)
- [ ] Achievement / badge
- [ ] Difficulty seçeneği (kolay/normal/zor)

### Won't Have (MVP)
- ❌ Multiplayer
- ❌ Backend / leaderboard online
- ❌ Mobile dokunmatik kontrol (v1.1)
- ❌ Çoklu dil (sadece TR)

## v1.1 — Polish (MVP sonrası)

- Dokunmatik kontrol (joystick UI mobil için)
- Daha fazla level
- İçerik genişleme (soru bankası büyütme)

## v2 — Genişleme (Eğer Can devam etmek isterse)

- Karakter customization
- Yan questler (mağazadan görev al)
- Boss çeşitliliği (mini-boss, final boss)
- Hikaye/cutscene

## v3 — Ürün İhtimali (Çok uzun vade)

- Online leaderboard (backend gerekli)
- Sınıf modu (öğretmen panel, öğrenci hesapları)
- Müfredat editörü (öğretmen kendi sorularını yükler)
